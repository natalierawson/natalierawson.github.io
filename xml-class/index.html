XML Textbook Tutorial Index

These exercises wre completed during Spring 2016 for my Intro to XML class with UMass Lowell. The source for these tutorials was New Perspectives on XML Comprehensive 3rd Edition by Patrick M. Carey.

The goals for the tutorials were to:

  1. Demonstrate comfort with writing an XML document, including DTDs and schemas.
  2. Demonstrate comfort commuting XML documents to HTML documents with XSLT and XPath.
  3. Commute XML documents to HTML documents using saxxon on the command line.
  4. Demonstrate comfort creating XML documents using queries with XQuery.
