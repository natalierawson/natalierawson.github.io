Duxbury Free Library Database Page

This pages was designed and completed over the course of Summer 2016. I worked with the Duxbury Free Library to address goals for their current site. Specifically, an interest was expressed in merging their reference and database pages.

The goals for the project were to:

  1. Update the structure of the page so that it adheres to modern web standards.
  2. Implement an extensive dropdown menu for easier user access to databses.
  3. Organize the available resources in a more accessible way.
  4. Create buttons to direct users to commonly desired tasks.
  5. Create an accordion-like structure on the page for users to access the most popular resources.
  6. Create a more welcoming color-scheme based on the original site design.
