This project was for my Javascript and jQuery class at UMass Lowell in Summer 2016. The assignment was to create a website on any topic, so I chose to make a website reviewing musicals.

The goals for the assignment were to:

  1. Use rollover images (I used both CSS and jQuery solutions).
  2. Include a form.
  3. Implement a dropdown menu.
